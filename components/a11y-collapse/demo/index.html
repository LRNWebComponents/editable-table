a11y-collapse demo

By default, the clicking the icon button triggers expand/collapse.

a11y-collapse demo, with accordion attribute

With the accordion attribute, the clicking anywhere on the heading triggers expand/collapse.

a11y-collapse-group, with radio attribute

With the radio attribute, only one item can be expanded at a time.

a11y-collapse-groups demo with global options

The following options can be set in the global-options attribute to apply to all a11y-collapse items in a group: icon, icon-expanded, label, label-expanded, tooltip, tooltip-expanded.
